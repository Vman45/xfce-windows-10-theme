# XFCE Windows 10 Theme

## Features

* Backgrounds
* Lock screens
* Cursors
* Icons (thanks to [B00merang](https://github.com/B00merang-Artwork/Windows-10))
* Task bar
* Window appearance
* Keyboard shortcuts

## Installing

```bash
curl -sL https://github.com/darkwebdesign/xfce-windows-10-theme/blob/master/install.bash | sudo -E bash -
```
